Optimizador de Préstamos

Modelo de Programación Lineal Entera Mixta que maximiza el aprovechamiento de tu tiempo de lectura con 9 restricciones activas.

Modelo matemático – MILP Extendido
Variables:  xⱼ ∈ {0,1}  ·  hⱼ ≥ 0  ·  yⱼ ∈ {0,1}
Max Z = ∑ [ wⱼ · hⱼ  +  B · yⱼ ]
  wⱼ = 1 + 0.4·(catⱼ == pref) − 0.15·dificultadⱼ
  B  = 5.0  (bonus por completar libro)
R1 ∑ hⱼ ≤ T_h
R2 hⱼ ≤ Hⱼ · xⱼ ∀j
R3 ∑ xⱼ ≤ M
R4 ∑_{cat==pref} xⱼ ≥ 1
R5 Hⱼ · yⱼ − hⱼ ≤ 0 ∀j
R6 yⱼ ≤ xⱼ ∀j
R7 ∑ hⱼ ≥ 0.4 · T_h
R8 hⱼ ≥ 0.2 · Hⱼ · xⱼ ∀j
R9 xⱼ, yⱼ ∈ {0,1} hⱼ ≥ 0
